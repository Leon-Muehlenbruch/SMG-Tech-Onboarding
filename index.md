# SMG Tech-Onboarding

Willkommen. Diese Doku beschreibt Schritt für Schritt, wie ein **Mac-Laptop** für die Arbeit bei der SawatzkiMühlenbruch GmbH eingerichtet wird.

Sie richtet sich an zwei Zielgruppen:

1. **Techniker:innen**, die das Setup durchführen → folgen den vier Workflow-Kapiteln in der angegebenen Reihenfolge.
2. **Mitarbeitende**, deren Laptop eingerichtet wurde → können in der **Software-Referenz** nachlesen, wozu jedes Tool dient.

---

## Voraussetzungen

Bevor du loslegen kannst, brauchst du:

- ✅ einen funktionsfähigen Mac (macOS aktuell)
- ✅ einen Browser (Safari reicht für den Start)
- ✅ eine **Firmen-E-Mail-Adresse** der SawatzkiMühlenbruch GmbH
- ✅ eine **persönliche E-Mail-Adresse** (für den Claude-Account)

Außerdem muss der **Admin** vorher folgende Einladungen rausschicken:

- GitHub-Org-Einladung an die Firmen-E-Mail
- Claude-Org-Hinzufügung an die persönliche E-Mail

---

## Der Workflow auf einen Blick

| # | Kapitel | Was passiert hier? |
|---|---|---|
| 1 | [Vorab](01-vorab.html) | Arbeitsordner anlegen, Node.js & Command Line Tools installieren |
| 2 | [GitHub](02-github.html) | Account erstellen, GitHub Desktop installieren, Test-Repo klonen |
| 3 | [Claude KI](03-claude.html) | Account erstellen, Claude Desktop installieren, MCPs einrichten |
| 4 | [Test & erste Nutzung](04-test-und-erste-nutzung.html) | Verbindungen prüfen, Test-Aufgabe ausführen, Push verifizieren |

Plane für die komplette Einrichtung **etwa 60–90 Minuten** ein.
