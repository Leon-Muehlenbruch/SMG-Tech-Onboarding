# SMG Tech-Onboarding

---

## Voraussetzungen

Bevor du loslegen kannst, brauchst du:

- einen funktionsfähigen Mac (macOS aktuell)
- einen Browser (Safari reicht für den Start)
- eine **Firmen-E-Mail-Adresse** unter `@schickma.de` (wird sowohl für GitHub als auch für Claude verwendet)

Außerdem muss der **Admin** vorher folgende Einladungen rausschicken:

- GitHub-Org-Einladung an die Firmen-E-Mail
- Claude-Org-Hinzufügung an die Firmen-E-Mail
