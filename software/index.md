# Software-Referenz

Detailseiten zu allen Tools, die im Onboarding installiert werden. Diese Seiten sind primär für **Mitarbeitende** gedacht, die nach dem Setup wissen wollen, was eigentlich auf ihrem Rechner läuft und warum.

## Übersicht

| Tool | Wofür? | Wo installiert? |
|---|---|---|
| [Node.js](nodejs.html) | JavaScript-Laufzeit, Voraussetzung für viele Tools/MCPs | systemweit |
| [Command Line Tools](command-line-tools.html) | `git`, Compiler, Basis-Werkzeuge fürs Terminal | systemweit |
| [GitHub Desktop](github-desktop.html) | Grafische Oberfläche für Git/GitHub | App im Programme-Ordner |
| [Claude Desktop](claude-desktop.html) | Native KI-App (Chat + Code) | App im Programme-Ordner |
| [Filesystem MCP](filesystem-mcp.html) | Gibt Claude Zugriff auf den Arbeitsordner | Konnektor in Claude Desktop |
| [GitHub Connection](github-connection.html) | Gibt Claude Zugriff auf GitHub-Repos | Konnektor in Claude Desktop |
| [GitHub CLI (`gh`)](github-cli.html) | Authentifiziert den Mac gegenüber GitHub (für `git push`) | systemweit |
| [Vercel (+ V0)](vercel.html) | Hosting-Plattform, deployt Sites automatisch bei jedem Push | Web + Konnektor in Claude Desktop |

Die Reihenfolge in der Sidebar entspricht der Installationsreihenfolge im Workflow.
